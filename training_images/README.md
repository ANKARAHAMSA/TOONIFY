# TOONIFY — Custom LoRA Training Datasets

This directory contains the curated training datasets used for fine-tuning custom LoRA styles in **TOONIFY**.

## Datasets Included
- `arcane/`: Curated dataset for Arcane Painterly style LoRA training (~75 images + captions)
- `claymation/`: Dataset for 3D Claymation stop-motion style LoRA training

## Caption Format
Each image has a corresponding `.txt` file or `metadata.jsonl` entry with the trigger word `in arcane_toonify style`.
